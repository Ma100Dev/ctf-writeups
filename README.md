### The repository format
should roughly follow the format described below.
```
├── somectfSomeyear
│   ├── README.md (general CTF info)
│   └── category1
│       ├── chall1
│       │   └── chall1.md (the actual writeup)
│       └── chall2
│           ├── exploit_script.py
│           ├── extraFile.js
│           └── chall2.md
├── LICENSE
└── README.md (this file)
```
